# Private Equity Deal Intelligence Platform

An end-to-end analytics platform that screens potential buyout targets using financial metrics, deal scoring models, and LBO return simulations.

## Live Demo

[Open the Dashboard](https://pe-deal-intelligence-platform-9xcgmvmdkzxuzmlx4kvq8j.streamlit.app/)

## Project Architecture

Python Synthetic Data Generator
↓
Snowflake Data Warehouse
↓
SQL Financial Metrics
↓
Deal Scoring Engine
↓
LBO Return Simulation
↓
IRR Analysis
↓
Streamlit Investment Dashboard

## Key Features

* Financial metrics modelling (EBITDA margin, leverage, growth)
* Deal scoring algorithm
* LBO return simulation
* IRR-based investment ranking
* Interactive investment screening dashboard

## Technology Stack

* Python
* Snowflake
* SQL
